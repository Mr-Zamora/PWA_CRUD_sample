About the Recipe Catalogue

This Recipe Catalogue is a sample web development project for software engineering students. The design and functionality serve as a guide to understanding web technologies and UI/UX principles for responsive and user-friendly web applications.

Tech Stack

  • Frontend: HTML5, CSS (Flexbox & Grid)
  • Backend: Flask (Python)
  • Database: JSON for data storage (extendable to SQL databases)
  • Responsive Design: Media queries, CSS grid, and flexbox for layout optimisation across devices

Design Principles

The UI/UX design follows modern web principles to ensure a seamless user experience. Key design considerations include:

  • Easy-to-navigate interfaces
  • Minimalistic design for clarity and focus
  • Optimised user flows for efficient interactions
  • Accessibility for all users

Progressive Web Application

This project will be developed as a Progressive Web Application (PWA), enabling offline capabilities and an app-like experience. The Flask backend will handle the server-side logic, and client-side interactions will be managed using JavaScript.

Project Goals

This project teaches students the integration of various web development technologies, including:

  • Building multi-page web applications using Flask
  • Creating a responsive UI with HTML, CSS, and JavaScript
  • Handling CRUD operations (Create, Read, Update, Delete) with JSON-based data
  • Documenting functionality using flowcharts and diagrams
  • Deploying the application on a live server (e.g., PythonAnywhere)
