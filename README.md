# Filipino Recipes - PWA UI Sample

A responsive, accessible web application for browsing and sharing Filipino recipes. This project serves as an educational example for Year 12 Software Engineering students learning web development fundamentals.

## Project Overview

This project demonstrates modern web development practices including:

- Semantic HTML5 structure
- Responsive CSS layouts using Flexbox and Grid
- Accessibility best practices
- Progressive enhancement
- Mobile-first design approach

## Features

- Browse recipe cards on the home page
- View detailed recipe information
- Add new recipes via form
- Mobile-responsive design
- Accessible navigation

## Educational Resources

This project includes several educational resources:

- **[CSS_ORGANIZATION.md](docs/CSS_ORGANIZATION.md)**: Guide for organizing CSS in web projects
- **[FLASK_PATHWAY.md](docs/FLASK_PATHWAY.md)**: Instructions for transitioning to a Flask-based application

## Future Development

This project is designed to be converted into a Progressive Web Application (PWA) using:

- Flask backend
- SQLite database
- Service workers for offline functionality
- JSON data storage
